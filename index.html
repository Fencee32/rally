Képzeljétek! A Rally hivatalos Magyar helyesírás szerint 1 l-lel írandó: Rali.

Github token generálás

  1. Profil → Settings
  2. Bal alsó sarok → developer settings
  3. Classic Token
  4. Megjegyzés, hogy mi ez a token és lejárati idő beállítása
  5. Mindent kipipálunk (biztos ami bizots) → Generate Token

Repo clone

  1. Github repo főoldalon katt a "Code" gombra → Copy url
  2. CMDben git clone <url>
  3. VSCode indítása: code <A létrejött mappa neve>

VSCode Sync

  1. Bal oldalon felülről a harmadik ikon: Source Control
